# Barra de progreso para tareas en Obsidian

Añade barras de progreso automáticas a tareas con subtareas en tus notas de Obsidian.
Hecho con Claude.ai

## Ejemplo

```markdown
- [ ] Tarea 1 ████████████░░░░░░░░░░░░
    - [x] Sub tarea 1-1
    - [x] Sub tarea 1-2
    - [ ] Sub tarea 1-3 ████████████████░░░░░░░░
        - [x] Sub sub tarea 3-1
        - [x] Sub sub tarea 3-2
        - [ ] Sub sub tarea 3-3
    - [ ] Subtarea 4
- [ ] Tarea 2 ████████████░░░░░░░░░░░░
    - [ ] Sub tarea 2-1
    - [x] Sub tarea 4-2
```

## Activar en una nota

Añadir propiedad barra en una nota. Opciones:

barra : 0|false -> plugin inactiva
barra : 1|true -> barra simple
barra : 2|porcentaje|percent -> barra con porcentaje
